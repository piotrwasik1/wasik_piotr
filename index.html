Możliwość formatowania tekstu w HTML

  • Pogrubienie [b] [/b]
  • Kursywa [i] [/i]
  • Podkreślenie [u] [/u]
  • Przekreślenie [strike] [/strike]
  • Indeks górny [sup] [/sup]
  • Indeks dolny [sub] [/sub]
  • Przejście do nowego wiersza w tekście [br]
  • zapobieganie automatycznemu przenoszeniu fragmentu tekstu do nowego wiersza [nobr] [/nobr]
  • spacja nie łamliwa &: ndsp

Nagłówki w stronie WWW

  1. [h1] bedzie najwiekszym nagłówkiem
  2. [h2] ... [h3] Będą coraz większe

Możliwości funkcji font

  1. [font size = "wielkosc czcionki"]
  2. [font color = "kolor czcionki" ]
  3. [font face = "rodzaj czcionki" ]

3 najprostrze rodzje kolorów do użycia funkcji font

  3. Hexadecymalny, #RRGGBB : #FF00000-czerwony , #00FF00-zielony , #0000FF-niebieski

2. Populrne interfejsy urządzeń peryferyjnych do komputera

  1. Interfejsy na przesył obrazu
    1. HDMI
      • Type A-Standard
      • Type B-Dual-Link
      • Type C-Mini
      • Type D-Micro
      • Type E-Zawiera blokującą ochronę aby kabel Typu Standard się nie ruszał
    2. DVI
      • DVI-I- Przesyła dane cyfrowe i analogowe
      • DVI-D- Przesyła dane cyfrowe
      • DVI-A- Przesyła dane analogowe
    3. D-SUB
      • DE-9- Standard
      • DE-9M- Standard bez śrub mocujących
      • DA-15- Złącze GamePort
      • DA-15F- Złącze VGA,SVGA i XGA
  2. Interfejsy na przesył danych
    1. Jack
      • Jack 6,3mm
      • Jack 3,5mm(minijack)
      • Jack 2,5mm(microjack)
  3. Interfejsy na przesył danych
    1. USB
      • Type A (USB1.0 & 2.0)
      • Type B (USB1.0 & 2.0)
      • Mini A (USB 2.0)
      • Mini B (USB 2.0)
      • Mini AB (USB 2.0)
